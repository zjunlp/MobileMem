# `templates/` — HTML Render Templates

HTML templates that are rendered to images for the screenshot-based image
categories. Generators fill in the placeholder fields (and, for chats, the
message DOM) and rasterize the page to a PNG.

| Template(s) | Used for |
|-------------|----------|
| `Kindle.html`, `微信读书.html` | Book / reading screenshots |
| `Spotify.html`, `网易云音乐.html` | Music screenshots |
| `YouTube.html`, `B站.html` | Video screenshots |
| `Amazon.html`, `淘宝订单.html` | Shopping screenshots |
| `TrainTicket.html`, `火车票.html` | Ticket screenshots |
| `PaymentTransfer.html`, `微信转账.html` | Payment / transfer screenshots |
| `wechat_group.html`, `telegram_group.html`, `discord_group.html`, `x_dm.html` | Group-chat screenshots |
| `friend.html`, `twitter_feed.html` | Social-media posts |
